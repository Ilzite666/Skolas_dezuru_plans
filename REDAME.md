# 🏫 Skolotāju dežūru plānošanas sistēma

Šī ir interaktīva sistēma, kas izveidota Rīgas Pārdaugavas pamatskolas vajadzībām, lai atvieglotu iknedēļas dežūru grafika sastādīšanu.

## 🚀 Kā palaist programmu?
Programma ir pieejama tiešsaistē: [SPIED ŠEIT, LAI ATVĒRTU](https://ilzite666.github.io/Skolas_dezuru_plans/)

## 📖 Lietošanas instrukcija
1. **Datu automātika:** Sistēma automātiski seko līdzi, lai viens skolotājs netiktu ielikts divās vietās vienlaicīgi tajā pašā dienā.
2. **Automātiska saglabāšana:** Katra izmaiņa tiek uzreiz saglabāta Jūsu pārlūkprogrammā. Pat ja nejauši aizvērsiet lapu, dati nepazudīs.
3. **Arhīva veidošana:** Kad plāns ir gatavs, nospiediet **"Saglabāt versiju"**. Jūs varat jebkurā brīdī apskatīt iepriekšējo semestru vai nedēļu plānus arhīva izvēlnē.
4. **Drukāšana:** Poga **"Drukāt"** automātiski sagatavo plānu A4 ainavas (Landscape) formātā, lai tas būtu gatavs izvietošanai pie sienas.
5. **Skolotāju saraksta maiņa:** - Nospiediet zaļo pogu **"Rediģēt skolotāju sarakstu"**.
   - Atvērtajā Google Sheets tabulā veiciet izmaiņas (pievienojiet vai dzēsiet vārdus).
   - Aizveriet tabulu un pārlādējiet dežūru programmas lapu.

## 💻 Tehniskā informācija
- **Datu avots:** Google Sheets API (CSV formātā).
- **Tehnoloģijas:** HTML5, CSS3 (Print Media Queries), JavaScript (Async/Await, LocalStorage).
- **Izstrādātājs:** Ilze Jakovļeva (2026)
